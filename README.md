# Object-Oriented Programming in Java

This repository demonstrates core Object-Oriented Programming (OOP) concepts in Java with structured examples for learning and interview preparation.

## 📌 Concepts Covered

* Abstraction
* Encapsulation
* Inheritance
* Polymorphism
* Collections Framework
* Arrays
* Exception Handling
* Static and Instance Members
* Method Overloading and Overriding

## 📁 Project Structure

The project is organized into packages based on OOP concepts:

* `abstraction`
* `encapsulation`
* `inheritance`
* `polymorphism`
* `collections`
* `arrays`
* `exceptions`
* `strings`

Each package contains simple, well-structured examples to illustrate the concept clearly.

## 🎯 Purpose

* Practice Core Java fundamentals
* Strengthen OOP understanding
* Prepare for technical interviews
* Demonstrate clean project organization

## 🛠️ Technologies Used

* Java
* IntelliJ IDEA
* Git & GitHub

## 🚀 How to Run

1. Clone the repository
2. Open in IntelliJ IDEA or any Java IDE
3. Navigate to the desired package
4. Run the `Main` class inside the package
